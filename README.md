# Site de Streaming de Música

Este projeto é um site de streaming de música desenvolvido com **HTML5** e **CSS3**, criado como parte de um estudo prático sobre estruturação de páginas web, design responsivo e experiência do usuário.  
O layout foi planejado previamente em **Figma**, servindo como base para a implementação do design final no código.

## Tecnologias e Skills Aplicadas

- **Linguagens:** HTML5, CSS3  
- **Ferramenta de prototipação:** Figma
